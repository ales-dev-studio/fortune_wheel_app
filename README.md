## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ales-dev-studio/fortune_wheel_app.git
   ```

2. Change to the project directory:

   ```bash
   cd portfolio_app
   ```

3. Install the dependencies:

   ```bash
   flutter pub get
   ```

4. Run the app:

   ```bash
   flutter run
   ```
